# OJT Team Portfolio

A data-driven agency landing page introducing our team and showcasing
all five capstone projects. Built with vanilla HTML, CSS, and JavaScript
as part of the On-Job Training Advanced JavaScript Capstone.

## Live Projects
| Project | Repo | Live |
|---|---|---|
| Quiz App | [ojt-quiz](https://github.com/Avishkar-05/Quiz-App.git) | [Live](https://avishkar-05.github.io/Quiz-App/) |
| Expense Tracker | [ojt-expense](https://github.com/Adityadolhare01/Expense-Tracker.git) | [Live](https://cute-stardust-dfd388.netlify.app/) |
| News Feed | [ojt-news](https://github.com/Avishkar-05/news-app.git) | [Live](https://avishkar-05.github.io/news-app/) |
| GitHub Explorer | [ojt-github-explorer](https://github.com/kaleabhayka-oss/GitHub-Explorer.git) | [Live](https://githubexplorer34.netlify.app/) |



## Features
- Team member cards with avatar, skills, and GitHub profile links
- Light / dark theme toggle persisted via localStorage
- Contact form with client-side validation
- Fully responsive from 320px mobile to desktop
